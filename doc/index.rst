ansi-colors
===========

Synopsis
--------

**ansi-colors** [*lower* *upper*]

Description
-----------

Print numbers from 0 to 150, and format using ANSI escape sequences
(specifically **\\e[m**).

*lower* *upper*

    Print numbers from *lower* to *upper*
