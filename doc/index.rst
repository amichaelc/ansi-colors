ansi-colors
===========

Synopsis
--------

**ansi-colors** [*upper* | *lower* *upper*]

Description
-----------

Print numbers from 0 to 150, and format using ANSI escape sequences (specifically **\\e[m**).

*upper*

    Print numbers from 0 to *upper*

*lower* *upper*

    Print numbers from *lower* to *upper*
